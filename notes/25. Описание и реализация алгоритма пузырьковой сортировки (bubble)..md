#### [[Вопросы к Экзамену]]
---
### Описание
Суть сортировки заключается в том, что на каждом шаге берутся два соседних элемента и располагаются в порядке возрастания. Это действие совершается с каждой парой, начиная с конца массива и вплоть до начала. Такой проход по массиву повторяется до тех пор, пока не останется необходимости в перестановках.
### Сложность
Худшее время: $O(n^2)$
В среднем: $O(n^2)$
Лучшее время: $O(n)$
### Реализация
``` python
def bubbleSort(a: list[int]):
	for i in range(1, len(a)):
		swapped = False
		for j in range(len(a) - 1, i, -1):
			if a[j] < a[j - 1]:
				a[j - 1], a[j] = a[j - 1], a[j] # swap a[j - 1] and a[j]
				swapped = True
		if not swapped:
			break
```
