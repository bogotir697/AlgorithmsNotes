#### [[Вопросы к Экзамену]]
---
### Описание
Самая простая сортировка, которая для каждой позиции в последовательности ищет минимальный элемент, после чего меняет элементы с индексами текущей позиции и найденного элемента:
### Сложность
Худшее время: $O(n^2)$
В среднем: $O(n^2)$
Лучшее время: $O(n)$
### Реализация
``` python
def selectionSort(a: list[int]):
	for i in range(len(a)):
		k = i
		for j in range(i + 1, len(a)):
			if a[j] < a[k]:
				k = j
		a[i], a[k] = a[k], a[i] # swap a[i] and a[k]
```
