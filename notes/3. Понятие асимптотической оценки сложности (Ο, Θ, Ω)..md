### Асимптотические функции
##### Асимптотически точная граница
$$ \exists c_1, c_2, n_0: f(n) = \Theta(g(n)) \Leftrightarrow c_1 g(n) \leq f(n) \leq c_2 g(n) \quad \forall n > n_0 $$
![[Theta(g(n)).png]]
##### Асимптотически верхняя граница
$$\exists c, n_0: f(n) = O(g(n)) \Leftrightarrow 0 \leq f(n) \leq cg(n) \quad \forall n > n_0$$
![[O(g(n)).png]]
##### Асимптотически нижняя граница
$$\exists c, n_0: f(n) = \Omega(g(n)) \Leftrightarrow 0 \leq cg(n) \leq f(n) \quad \forall n > n_0$$
![[Omega(g(n)).png]]

#### Свойства асимптотических функций
**Транзитивность:**
* Из $f(n) = \Theta(g(n))$ и $g(n) = \Theta(h(n))$ следует $f(n) = \Theta(h(n))$
* Из $f(n) = O(g(n))$ и $g(n) = O(h(n))$ следует $f(n) = O(h(n))$
* Из $f(n) = \Omega(g(n))$ и $g(n) = \Omega(h(n))$ следует $f(n) = \Omega(h(n))$

**Рефлексивность:**
* $f(n)=\Theta(f(n))$
* $f(n)=O(f(n))$
* $f(n)=\Omega(f(n))$

**Симметрия:**
* $f(n)=\Theta(g(n)) \Leftrightarrow g(n)=\Theta(f(n))$

**Перестановочная симметрия:**
* $f(n)=O(g(n))\Leftrightarrow g(n)=\Omega(f(n))$
