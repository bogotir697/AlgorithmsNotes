#### [[Вопросы к Экзамену]]
---
### Описание
Изначально отсортированная последовательность пуста; на каждом шаге из набора входных данных выбирается элемент и помещается на нужное место в уже отсортированной последовательности; шаги продолжаются до тех пор, пока набор входных данных не закончится.
### Сложность
Худшее время: $O(n^2)$
В среднем: $O(n^2)$
Лучшее время: $O(n)$
### Реализация
``` python
def insertionSort(a: list[int]):
	for i in range(1, len(a)):
		j = i
		while a[j] < a[j - 1] and j > 0:
			a[j - 1], a[j] = a[j], a[j - 1] # swap a[j - 1] and a[j]
			j -= 1
```
