### Линейный поиск
#### Описание
Проходимся по массиву. Если находим элемент, возвращаем его позицию. Если нет - значит элемента нет в массиве.
#### Сложность
Лучшее время: $O(1)$
В среднем: $O(n)$
#### Реализация
```python
def find(a: list[int], x: int) -> int:
	for i, elem in enumerate(a):
		if elem = x:
			return i
	return -1
```
### Бинарный поиск
#### Описание
Есть 2 указателя: на самый левый и на самый правый элементы массива. Находим средний элемент между ними. Если его значение больше нужного, то сдвигаем правый указатель на него, если нет - левый. Повторяем алгоритм до тех пор, пока два указателя не будут указывать на один и тот же элемент.
#### Сложность
Лучшее время: $O(1)$
В среднем: $O(\log_2{n})$
#### Реализация
```python
def binarySearch(a: list[int], x: int) -> int:
	l = 0
	r = len(a) - 1
	while l < r:
		m = (l + r) // 2
		if x > m:
			l = m
		elif x < m:
			r = m
		else:
			return m
	return l
```
