### Алфавит формального языка
> Множество атомарных (неделимых) символов какого-либо формального языка (иногда их называют буквами по аналогии с алфавитами естественных языков или символами). Из символов алфавита формального языка строятся слова, а заданием формальной грамматики — допустимые выражения языка. Чаще всего алфавит рассматривается как непустое конечное множество.

#### Примеры
$\{a,b,c,d ..., x, y, z\}$, $\{+, -, *, /\}$, $\{\sin, \cos, \tan, \log, \exp\}$.
### Простой (наивный) поиск
#### Описание
Суть алгоритма состоит в последовательном переборе с последующим сравнением символов строки и образца

**Сложность:** $O(n*m)$, где n — длина искомой строки, а m — длина образца.
#### Реализация
```python
def NaiveSearch(S: str, P: str) -> int:
	for i in range(len(S) - len(P) + 1):
		for j in range(len(P)):
			if S[j] != P[i + j]:
				break
		else:
			return i
	return -1
```
