#### [[Вопросы к Экзамену]]
---
### Описание
1. Найти ребро минимального веса
2. Проверить на основе *системы непересекающихся множеств*, создаёт ли ребо цикл. Если да, повторить шаг 1
3. Сохранить ребро и повторять алгоритм, пока не найдётся $n - 1$ ребро, где $n$ - это количество вершин графа
### Реализация
``` python
# Система непересекающихся множеств
def find(x: int, parent: list[int]):
	if parent[x] != x:
		parent[x] = find(parent[x], parent)
	return parent[x]

def union(x: int, y: int, parent: list[int], rank: list[int]):
	rootX = find(x, parent)
	rootY = find(y, parent)
	if rootX != rootY:
		if rank[rootX] > rank[rootY]:
			parent[rootY] = rootX
		elif rank[rootX] < rank[rootY]:
			parent[rootX] = rootY
		else:
			parent[rootY] = rootX
			rank[rootX] += 1
```
